Проект с автотестами для сайта Яндекс Самокат
https://qa-scooter.praktikum-services.ru/

Содержит проверки следующих сценариев:

- Выпадающий список в разделе «Вопросы о важном». 
  Проверка соответствующих текстов каждого вопроса и каждого ответа. 

- Заказ самоката. 
  Проверка всего флоу позитивного сценария с двумя наборами данных через разные точки входа: 
  кнопки «Заказать» вверху страницы и внизу.

- Проверка,всплывающего окна с сообщением об успешном создании заказа.

- Проверка редиректа по клику на логотип «Самоката».

- Проверка редиректа по клику на логотип Яндекса.

- Также данный проект содержит allure отчет о прохождении тестов
